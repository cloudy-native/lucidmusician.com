We prioritized plugins with explicit AU compatibility based on developer specs and DAW integrations (e.g., Logic Pro support). The list draws from recent 2025 reviews and avoids non-plugin tools (e.g., standalone apps or browser-based generators like Hookpad's Aria or AIVA).

Pricing reflects current standard rates (one-time purchases unless noted; check official sites for sales/discounts). Pros and cons are gathered from personal experience, user reviews, developer claims, and expert analyses. Not all plugins have exhaustive con lists, as some are highly regarded overall.

## Breakdown by Feature

Looking at the standout features of each plugin, we can group plugins into categories based on their primary focus and capabilities.

### Drag-and-Drop MIDI Export

Drag-and-drop MIDI export is a core workflow enhancer in chord generation plugins, allowing users to seamlessly transfer generated chord progressions, voicings, or full sequences directly into a DAW's timeline or piano roll without manual note-by-note entry. This feature streamlines composition by preserving timing, velocity, and note data, reducing friction between idea generation and production, and is especially valuable for iterative songwriting where rapid prototyping is key.

- **Scaler 3**: Supports exporting chord blocks and progressions with roman numeral notation for easy transposition in the DAW.
- **Captain Chords**: Enables drag-and-drop of entire progression builders, including integrated sections like melody or bass.
- **ChordJam**: Allows instant export of randomized rhythmic chord patterns for loop-based production.
- **Ripchord**: Facilitates quick export of triggered chord presets, ideal for layering in tracks.
- **AudioCipher MIDI Vault**: Exports text-derived progressions with tags for organized library management.
- **LANDR Composer**: Provides instant drag-export of AI-generated mood-based sequences.
- **InstaComposer 2**: Drags multi-track MIDI (chords, melody, etc.) directly to the DAW.
- **Spacewalk**: Offers drag-and-drop of evolving ambient progressions with humanized elements.

### Progression Builder/Sequencer
A progression builder or sequencer lets users construct, arrange, and loop chord sequences in a dedicated interface, often with timeline views or block-based editing. This expands beyond single chords to full harmonic structures, incorporating loop lengths, repeats, and transitions to mimic song sections like verse-chorus, fostering a narrative flow in compositions and aiding in genre-specific arrangements.

- **Scaler 3**: Features a multi-track sequencer for building and evolving progressions with voicings and inversions.
- **Captain Chords**: Includes a drag-and-drop builder with rhythm controls for hit-song-inspired sequences.
- **ChordJam**: Built-in sequencer generates rhythmic patterns from one-knob chord starts.
- **Cthulhu**: MIDI input remapping creates progression-like arpeggios and triggers.
- **InstaComposer 2**: Generates 5-track progressions locked to key and tempo for full ideas.
- **Liquid Music**: Layer system builds progressions via shape-drawing for chords and rhythms.
- **Spacewalk**: Evolves ambient progressions with editing, locking, and swapping for seamless sequences.

### Randomization/AI Generation
Randomization or AI generation introduces variability through algorithms that suggest or create chords based on parameters like probability, mood, or genre, preventing creative blocks by offering unexpected yet musically coherent options. This feature democratizes advanced harmony for beginners while providing pros with efficient ideation, often blending rule-based theory with machine learning for organic-feeling outputs.

- **ChordJam**: Velocity and inversion randomization tied to scale and groove presets.
- **AudioCipher MIDI Vault**: AI-driven text-to-chord with inversion and duration randomization.
- **MIDI Wizard 2.0**: AI generates chords/melodies by genre, key, and tempo, with progression detection.
- **LANDR Composer**: AI crafts mood-based (e.g., uplifting) progressions with style presets.
- **InstaComposer 2**: AI uses math/theory for probability-controlled multi-track generation.
- **dBdone AI Chords**: AI produces key/mood-based progressions with variation generation.
- **Harvest Mini**: Scale-locked randomization for chords and melodies with re-roll options.
- **World Chord Generator**: Voicing/inversion randomization on retriggers for variations.
- **Fluid Chords**: Randomizes scale-locked transitions with intelligent bending.

### Key/Scale Detection or Locking
Key and scale detection automatically analyzes input (MIDI or audio) to suggest or enforce harmonic constraints, while locking ensures all generated content stays diatonic to avoid dissonance. This foundational feature builds confidence in harmonic choices, supports modal exploration, and integrates with broader composition tools for cohesive tracks across sections.

- **Scaler 3**: Analyzes MIDI/audio for real-time key/scale detection.
- **Tonespace**: Smart scale-fitting with 21 scales and automatic in-key adherence.
- **ChordJam**: Locks generation to selected scales and chord types.
- **AudioCipher MIDI Vault**: Controls scale alongside chord type and BPM.
- **MIDI Wizard 2.0**: Generates by specified key with existing progression detection.
- **LANDR Composer**: Implicitly locks AI outputs to key for mood consistency.
- **InstaComposer 2**: Key/tempo lock for all AI-generated tracks.
- **Liquid Music**: In-key shape-drawing for melodies and chords.
- **Fluid Chords**: Scale-locked transitions between chords.
- **Harvest Mini**: Strictly scale-locked randomization to maintain tonality.
- **World Chord Generator**: Supports 57 scales/modes for global locking.

### Voicings/Inversions Controls
Voicings refer to note spacing and density within a chord (e.g., open vs. close), while inversions reorder notes for smoother bass lines or transitions. These controls allow fine-tuning for emotional nuance—dense for tension, sparse for airiness—enhancing playability on instruments and voice-leading in arrangements.

- **Scaler 3**: Progression builder includes detailed voicings and inversions.
- **Captain Chords**: Rhythm/voicing controls for customizable chord spreads.
- **ChordJam**: Inversion randomization with velocity tweaks.
- **Chordz**: Customizable chord types, voicings, inversions, and octaves.
- **AudioCipher MIDI Vault**: Inversion and randomization for text-based chords.
- **LANDR Composer**: MIDI editor adjusts voicings via notes and velocity.
- **dBdone AI Chords**: Editable voicings post-AI generation.
- **World Chord Generator**: Voicing/inversion randomization.
- **Spacewalk**: Humanized voicings via sparseness, suspension, and deviation controls.

Preset libraries are pre-curated collections of chords, progressions, or styles drawn from real songs, genres, or moods, serving as starting points for inspiration. They accelerate workflows by providing proven harmonic templates, customizable for originality, and often categorized for quick access in diverse musical contexts.

- **Scaler 3**: Genre/mood presets with thousands of chords/progressions.
- **Captain Chords**: 500+ presets from hit songs with roman numeral transposition.
- **ChordJam**: Groove presets focused on pop/EDM.
- **Cthulhu**: Preset chord banks for arpeggios and triggers.
- **Ripchord**: Import/export custom presets with collaborative sharing.
- **Tonespace**: 39 chord types across 21 scales as visual presets.
- **LANDR Composer**: Style presets for trap/EDM/pop.
- **ChordConverter**: Style presets for jazz/pop transformations.
- **Spacewalk**: 7 character presets for ambient evolution.

### Real-Time MIDI Input/Triggering
Real-time MIDI input processes live keyboard or controller data to generate or trigger chords instantly, enabling performative composition or live remixing. This interactivity bridges plugins with hardware, supporting improvisation and immediate feedback for dynamic sessions.

- **Captain Chords**: Real-time MIDI keyboard input for progression building.
- **Cthulhu**: Chord memory triggers full chords from single notes in real-time.
- **Chordz**: Keyboard-mapped triggering where each key plays a full chord.
- **Ripchord**: Single-note triggering of full chord presets, velocity-sensitive.
- **Tonespace**: MIDI input/output for synth triggering with grid clicks.
- **World Chord Generator**: Real-time MIDI triggering with scale support.
- **Pentacom ChordBloom**: Split-keyboard zones for root/type triggering.

### Chord Memory/Triggering from Single Notes
Chord memory stores and recalls full chords triggered by a single root note or key press, simplifying complex harmony playback without polyphonic input. This is ideal for monophonic controllers or quick sketching, promoting fluidity in idea capture.

- **Cthulhu**: Dedicated chord memory mode for single-note triggers.
- **Ripchord**: Preset-based single-note triggering.
- **Chordz**: Each key triggers a mapped full chord.
- **Pentacom ChordBloom**: Chord memory/recall via split-keyboard control.

Genre, mood, or style presets tailor generations to specific vibes (e.g., dark trap or uplifting pop), using curated algorithms or samples to evoke targeted emotions or sonic identities. This contextualizes harmony, helping users match progressions to project needs without starting from scratch.

Arpeggiators break chords into sequenced notes with rhythmic flair, while pattern tools add grooves or interpretations. These inject motion and energy, transforming static harmony into dynamic elements suitable for electronic or rhythmic genres.

### Editing Capabilities (Piano Roll, etc.)
